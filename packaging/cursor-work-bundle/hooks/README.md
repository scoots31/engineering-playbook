# Cursor hooks (examples in this bundle)

These files under `hooks/examples/` are meant to be **copied** into a product repository:

- `.cursor/hooks.json` at the product root (project hooks), or
- `~/.cursor/hooks.json` for user-wide hooks (paths relative to `~/.cursor/`).

See `hooks/examples/` for a destructive-command guard and merge notes.

**Verify** in Cursor: Hooks settings tab and Hooks output channel after saving `hooks.json`.

This work bundle does **not** include Claude Code hook configuration.
